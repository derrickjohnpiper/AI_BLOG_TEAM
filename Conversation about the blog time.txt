analyze_style:
  description: >
    Analyze the provided samples of Derrick-John's writing and create a detailed Style Profile.
    Include tone, sentence length, paragraph style, favorite phrases, structure preferences, and examples.

    A clear "Derrick-John Style Profile" document that can be used as reference for all future writing.

research_topic:
  description: >
    Research the topic: {topic}. Include fresh data, industry insights, company context, and potential angles.

  expected_output: >
    Comprehensive research notes with sources, key points, statistics, and examples.

How to Run It (Basic Python File)
In your main crew file (usually src/blog_writer/crew.py or similar), it should look something like this after the CLI scaffold:

from crewai.project import CrewBase, agent, crew, task
from crewai import Crew

@CrewBase
class BlogWriter:
    """Blog writing crew powered by your style"""

    agents_config = 'config/agents.yaml'
    tasks_config = 'config/tasks.yaml'

    @crew
    def crew(self) -> Crew:
        return Crew(
            agents=self.agents,
            tasks=self.tasks,
            process="sequential",   # or "hierarchical"
            verbose=2,
        )
